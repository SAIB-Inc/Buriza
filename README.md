# Buriza: A Cross-Platform Cardano Wallet Suite 🚀  
**Funded by Project Catalyst Fund 13**  

---

## Overview
**Buriza** is an ambitious, fully decentralized, open-source **Cardano** wallet project spanning multiple platforms—desktop, mobile, and browser. This monorepo houses all Buriza implementations, enabling a **consistent user experience** and **strong security** across devices. By giving users the choice to self-host and configure their own backend, **Buriza** ensures maximum control and fosters a truly decentralized ecosystem.  

---

## Problem Statement
Despite the variety of Cardano wallets, key issues persist:

- **Centralized Dependencies** ⚙️  
  Reliance on third-party servers introduces security and reliability risks.

- **Limited User Control** ⚠️  
  Many wallets don’t allow users to customize and self-host, undermining decentralization.

- **Platform Gaps** 🖥️📱  
  Few solutions provide a consistent, high-quality experience across desktop, mobile, and browser.

**Buriza** addresses these challenges with a unified, open-source, and community-driven solution.

---

## Key Features

1. **Cross-Platform Compatibility** 🌐  
   - **Desktop App**: Standalone, heightened security, optional full-node functionality.  
   - **Mobile Version**: On-the-go asset management, dApp integration, user-friendly UI.  
   - **Browser Extension**: Quick access for decentralized applications, with planned security enhancements.

2. **Full Node Capability (Desktop)** 💻  
   - For users seeking complete control and trustless transaction verification.

3. **Self-Host & Backend Flexibility** ⚙️  
   - Choose or configure your preferred backend servers, or run entirely on your own infrastructure.

4. **Integrated dApp Interaction** 🔗  
   - Desktop includes a built-in browser; mobile and browser extensions also support dApp integration for seamless usage.

5. **Enhanced User Experience** ✨  
   - Clear transaction dialogs, easy NFT/media rendering, and organized transaction histories across all versions.

6. **Open Source Transparency** 📝  
   - Licensed under Apache 2.0, fostering community trust, audits, and contributions.
